# 제미나이

## 역할

- [[프로필]]이 종종 사용하는 AI 도구.
- 원문 표기: Gemini
- [[AI 에이전트]]와 [[AI 정보 추적]]에서 추적할 대상.
- [[YouTube Gemini Summarizer]]에서 YouTube 영상 요약 API로 사용.
- 가격이 싸고 가성비가 좋아 정보 찾는 용도로 많이 사용.
- 특히 3.5flash가 성능 대비 사용 가능량이 많다고 평가함.

## 구독

- 월 11,000원 요금제를 사용 중.
- 관련 노트: [[정기 구독]]

## 장점

- 가격이 저렴함.
- 가성비가 좋음.
- 사용 가능량이 많아 정보 탐색용으로 부담이 적음.
- 빠르게 여러 정보를 찾아보는 용도에 적합함.

## 단점

- 다른 AI에 비해 긍정편향이 심한 편.
- 오류가 있음.
- 최대 지능은 [[코덱스]]나 Claude Opus 계열에 비해 낮다고 느낌.

## 사용 판단

- 정보 찾기, 넓게 훑기, 가성비 중심 작업에 적합.
- 중요한 판단, 고난도 추론, 정확도가 중요한 코딩/분석은 [[코덱스]]나 [[클로드]]와 교차검증하는 편이 좋음.

## 앞으로 채울 항목

- 주 사용 목적
- [[GPT]], [[클로드]]와 비교

## 메모

- 사용자가 말한 `3.5flash` 표기를 그대로 기록함. 정확한 공식 모델명은 나중에 필요하면 확인.
