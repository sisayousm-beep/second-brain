# ATLAS

## 상태

- 경제와 문명형 시뮬레이션을 결합한 게임 프로젝트.
- 기술 구현은 어느 정도 진행됨.
- 현재는 비주얼 작업이 필요한 단계.
- 관련 인물: [[프로필]]
- 관련 노트: [[프로젝트]], [[GitHub]], [[AI 에이전트]]

## 게임 콘셉트

- 경제, 정치, 문명 발전을 기반으로 국가나 기업을 키워가는 방향의 게임.
- 플레이어는 국가 또는 기업 같은 큰 단위의 주체를 성장시키는 것을 목표로 함.
- 핵심 재미는 자원, 경제 구조, 정치적 선택, 성장 전략을 조합해 장기적으로 세력을 키우는 데 있음.

## 현재 진행 상황

- 기술적인 구현은 어느 정도 완료.
- 앞으로는 화면 구성, UI, 시각적 정체성, 게임 내 정보 표현 같은 비주얼 작업이 중요함.

## 다음 할 일

- 메인 화면, 지도/도시/기업 관리 화면 등 핵심 UI 구조 잡기
- 경제ㆍ정치 정보를 한눈에 볼 수 있는 대시보드 설계
- 국가/기업 성장 과정을 시각적으로 보여주는 요소 정리
