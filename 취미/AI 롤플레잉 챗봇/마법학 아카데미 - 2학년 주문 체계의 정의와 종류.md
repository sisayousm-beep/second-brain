# 마법학 아카데미 - 2학년 주문 체계의 정의와 종류

상위 문서: [[마법학 아카데미 - 2학년 커리큘럼]]

## 과목 목적

주문 체계의 정의와 종류는 1학년에서 배운 영창논리율을 더 세분화해서 배우는 과목이다.

1학년에서는 영창의 기본 흐름을 배웠다면, 2학년에서는 영창논리율이 어떤 부분으로 구성되는지 배운다.

## 1단원. 영창논리율의 구조

영창논리율은 크게 두 부분으로 나뉜다.

- 실현부
- 논리부

실현부는 주문이 식을 어떤 방식으로 실현하는지를 다룬다.

논리부는 주문이 어떤 판단 체계 위에서 작동하는지를 다룬다.

## 실현부

### 2단원. 직렬실현

직렬실현은 식을 순서대로 실현하는 방식이다.

```text
A -> B -> C
```

하나의 과정이 끝난 뒤 다음 과정으로 넘어간다.

직렬실현은 안정성이 높지만, 실현 속도가 느릴 수 있다.

### 3단원. 병렬실현

병렬실현은 여러 식을 동시에 실현하는 방식이다.

```text
A
B -> C
D
```

여러 과정이 동시에 진행되므로 빠르고 강력할 수 있다.

다만 직렬실현보다 제어가 어렵고, 기초 단계에서는 실패 가능성이 더 높다.

## 논리부

### 4단원. 이원논리

이원논리는 주문의 상태를 두 가지로 나누는 방식이다.

```text
참 / 거짓
```

또는 1학년에서 배운 방식으로는 다음과 같다.

```text
실현됨 / 실현되지 않음
```

이원논리는 가장 기본적이고 안정적인 논리 체계다.

이원논리는 판단 상태가 단순하기 때문에 안정성이 높고 다루기 편리하다. 다만 상태 전환을 보수적으로 처리하므로 복잡한 주문에서는 속도가 느릴 수 있다.

### 5단원. 삼원논리

삼원논리는 주문의 상태를 세 가지로 나누는 방식이다.

```text
참 / 거짓 / 미정
```

여기서 미정은 아직 실현 여부가 확정되지 않은 상태를 의미한다.

삼원논리는 복잡한 주문 체계를 다룰 때 필요하지만, 2학년에서는 기초 개념만 다룬다.

삼원논리는 미정 상태를 활용할 수 있기 때문에 빠르고 고차원 마법에 유용하다. 다만 스칼라 마법이나 3차원 이하의 마법에서는 판단 상태가 불필요하게 늘어나 오히려 비효율적일 수 있다.

## 논리부 비교

| 논리 체계 | 상태 | 장점 | 한계 |
| --- | --- | --- | --- |
| 이원논리 | 참 / 거짓 | 안정성이 높고 편리함 | 복잡한 주문에서는 느릴 수 있음 |
| 삼원논리 | 참 / 거짓 / 미정 | 빠르고 고차원 마법에 유용함 | 스칼라 마법이나 3차원 이하의 마법에서는 비효율적일 수 있음 |

## 평가 방식

### 필기

- 실현부와 논리부의 차이 설명
- 직렬실현과 병렬실현 구분
- 이원논리와 삼원논리 설명
- 이원논리와 삼원논리의 장단점 비교

### 실습

- 간단한 주문을 직렬실현 구조로 작성
- 간단한 주문을 병렬실현 구조로 분류
- 주문 상태를 이원논리 또는 삼원논리로 판별
- 주문의 차원과 성격에 맞는 논리 체계 선택

## 챗봇 사용 기준

- 2학년 캐릭터는 영창논리율을 실현부와 논리부로 나누어 설명할 수 있다.
- 이원논리는 느리지만 안정성이 높고 편리한 논리 체계로 설명한다.
- 삼원논리는 빠르고 고차원 마법에 유용하지만, 스칼라 마법이나 3차원 이하의 마법에서는 비효율적일 수 있다고 설명한다.
- 고차원 마법의 구체적인 응용은 고학년 과정으로 안내한다.
