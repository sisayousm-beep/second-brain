# 마법학 아카데미 - 1학년 기초식이론

상위 문서: [[마법학 아카데미 - 1학년 커리큘럼]]

## 과목 목적

기초식이론은 식의 기본 개념과 작성법을 배우는 과목이다.

학생은 이 과목을 통해 식이 무엇인지 이해하고, 가장 기초적인 방식으로 식을 작성할 수 있어야 한다.

## 1단원. 식의 기본 개념

식은 마력을 구성하는 벡터공간, 혹은 그 원소다.

식은 마법의 내용에 해당하며, 주문을 통해 실현될 수 있는 대상이다.

기초식이론에서는 식을 복잡한 마법 현상으로 다루기보다, 마력을 표현하는 가장 기본적인 형식으로 다룬다.

## 2단원. 식의 작성법

식은 하나의 방식으로만 작성되지 않는다. 기초식이론에서는 여러 형식의 식 작성법을 배운다.

| 작성 방식 | 예시 | 설명 |
| --- | --- | --- |
| 단순합 | $A + B = C$ | 여러 항을 더해 하나의 식을 구성하는 가장 기본적인 방식 |
| 급수 | $\sum a_n$ | 반복되거나 누적되는 식을 표현할 때 사용하는 방식 |
| 행렬곱 | $Mv$ | 식의 배열, 변환, 관계를 표현할 때 사용하는 방식 |
| 선형결합 | $\alpha A + \beta B + \gamma C$ | 여러 식에 계수나 가중치를 부여해 하나의 식을 구성하는 방식 |

선형결합은 이후 2학년의 식의 최적화이론과 연결된다.

## 3단원. 식의 오류

기초식이론에서는 초보자가 자주 만드는 잘못된 식을 다룬다.

1학년 수준에서는 오류를 완전히 증명하기보다, 식이 제대로 작성되었는지 판별하는 데 집중한다.

### 계수가 정의되지 않은 식

식 안에 사용된 계수가 명확하게 정의되지 않은 경우다.

예를 들어 $\alpha A + B$에서 $\alpha$가 어떤 값인지, 어떤 조건에서 결정되는지 정의되지 않았다면 식은 불완전하다.

## 평가 방식

### 필기

- 식의 기본 개념 설명
- 식의 작성 방식 구분
- 식의 오류 판별

### 실습

- 단순한 식 작성
- 단순합, 급수, 행렬곱, 선형결합의 기초 표현
- 주어진 조건에 맞는 식 구성

## 챗봇 사용 기준

- 1학년 캐릭터는 식을 엄밀하게 증명하기보다 기본 개념과 작성 방식 중심으로 설명한다.
- 식의 오류를 말할 때는 "왜 틀렸는가"보다 "어떤 정의가 빠졌는가"를 우선 지적한다.
- 식의 최적화, 기저, 차원은 2학년 과정으로 안내한다.
