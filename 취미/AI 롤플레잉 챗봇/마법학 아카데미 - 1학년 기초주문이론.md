# 마법학 아카데미 - 1학년 기초주문이론

상위 문서: [[마법학 아카데미 - 1학년 커리큘럼]]

## 과목 목적

기초주문이론은 주문의 기본 개념과 영창의 역할을 배우는 과목이다.

학생은 이 과목을 통해 주문과 영창의 차이를 이해하고, 가장 기본적인 영창논리율에 따라 단순한 주문을 구성할 수 있어야 한다.

## 1단원. 주문의 기본 개념

주문은 식을 실현하는 체계다.

식이 마법의 내용이라면, 주문은 그 식을 실현시키는 체계다.

주문은 반드시 영창의 형태를 가질 필요는 없다. 영창은 주문의 한 형태일 뿐이며, 주문과 영창은 같은 것이 아니다.

## 2단원. 영창의 정의와 필요성

영창은 주문의 한 형태다.

영창은 식을 실현하기 위해 내용을 순서화하고, 결과를 정의한 뒤, 마법으로 실현하는 방식이다.

초보자에게 영창이 중요한 이유는 다음과 같다.

- 식의 선언을 명확하게 하기 위해
- 결과 정의를 분명하게 하기 위해
- 마법 실현 과정을 안정적으로 구성하기 위해

단, 영창은 주문 전체를 의미하지 않는다. 주문은 식을 실현하는 체계이고, 영창은 그 체계를 구성하는 한 방식이다.

## 3단원. 영창의 방법론

기초주문이론에서 다루는 영창의 기본 방법론은 다음 흐름을 따른다.

```text
식의 선언 -> 결과 정의 -> 마법 실현
```

| 단계    | 역할                    | 예시              |
| ----- | --------------------- | --------------- |
| 식의 선언 | 실현할 식을 선언한다.          | 식 A를 선언한다.      |
| 결과 정의 | 식이 실현되었을 때의 결과를 정의한다. | A의 결과를 C로 정의한다. |
| 마법 실현 | 정의된 결과를 마법으로 실현한다.    | C를 실현한다.        |

## 4단원. 보편적 영창논리율

1학년에서는 가장 보편적인 영창논리율만 배운다.

### 정의된 식만 실현될 수 있다

정의되지 않은 식은 영창을 통해 실현될 수 없다.

```text
정의된 식 -> 실현 가능
정의되지 않은 식 -> 실현 불가
```

### 시작된 주문은 반드시 종결되어야 한다

주문은 시작과 끝을 가져야 한다.

종결되지 않은 주문은 완성된 주문으로 취급되지 않는다.

```text
주문 시작 -> 주문 종결
```

### 주문은 실현된 상태와 실현되지 않은 상태로 나뉜다

이는 이원논리의 가장 기초적인 형태다.

```text
실현됨 / 실현되지 않음
```

1학년에서는 이 내용을 통해 주문의 상태를 가장 단순한 방식으로 구분한다.

## 5단원. 기초 주문 작성

기초 주문 작성은 다음 방법론을 따른다.

```text
식의 선언 -> 결과 정의 -> 마법 실현
```

예를 들어 식이 $A + B = C$라면, 기초 주문은 다음처럼 구성된다.

```text
식 A + B를 선언한다.
그 결과를 C로 정의한다.
C를 마법으로 실현한다.
```

이 단계의 목표는 강한 마법을 사용하는 것이 아니라, 식과 결과와 실현 과정이 명확하게 이어지는 주문을 작성하는 것이다.

## 평가 방식

### 필기

- 주문의 정의 설명
- 주문과 영창의 차이 설명
- 영창의 필요성 설명
- 보편적 영창논리율 설명

### 실습

- 주어진 식에 맞는 기초 주문 작성
- 식의 선언, 결과 정의, 마법 실현 구분
- 주문의 실현 상태 판별

## 챗봇 사용 기준

- 1학년 캐릭터는 주문을 영창 문구 자체가 아니라 식을 실현하는 체계로 설명한다.
- 주문과 영창을 같은 개념으로 설명하지 않는다.
- 복잡한 실현부, 논리부, 삼원논리는 2학년 과정으로 안내한다.
